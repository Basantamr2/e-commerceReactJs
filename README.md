# E-Commerce Web Application

A responsive E-Commerce web application built using React.js with real API integration.

## Features
- User Authentication (Register / Login)
- Product Listing & Filtering
- Shopping Cart Functionality
- Global State Management using Context API
- Responsive Design

## Technologies Used
- React.js
- JavaScript
- Context API
- REST API
- CSS3

## How to Run
npm install  
npm start
